AppVault позволяет скрывать, отключать и удалять любые приложения на вашем Android-устройстве — без root.

ОСНОВНЫЕ ФУНКЦИИ

- Скрытие приложений: Сделайте любое приложение невидимым
- Удаление приложений: Удалите ненужные приложения для текущего пользователя
- Без root: Работает через Shizuku с разрешениями ADB
- Восстановление в любой момент: Скрытые приложения можно снова сделать видимыми одним нажатием
- Поиск и фильтр: Быстро находите приложения по имени или имени пакета
- Системные приложения: Просматривайте и управляйте предустановленными приложениями
- Современная тёмная тема: Чистый, минимальный дизайн
- Мультиязычность: Поддержка 11 языков

КАК ЭТО РАБОТАЕТ

AppVault использует Shizuku для выполнения ADB-команд прямо с телефона. Установите Shizuku из Play Store, запустите один раз через ADB, и всё готово.

БЕЗОПАСНО И ОБРАТИМО

Все изменения обратимы. Скрытые приложения не удаляются — они просто отключаются для вашего профиля.

ПРИМЕЧАНИЕ: Требуется приложение Shizuku (бесплатно в Play Store). Необходима однократная настройка ADB.
